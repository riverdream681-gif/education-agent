# 高中数学 - 解析几何

---

## 直线的倾斜角与斜率

**倾斜角**：直线向上的方向与 x 轴正方向所成的最小正角 α，范围 [0, π)。  
**斜率**：k = tan α (α ≠ π/2)。  
**两点式斜率公式**：过 P₁(x₁,y₁), P₂(x₂,y₂) (x₁≠x₂) 的直线斜率 k = (y₂ - y₁)/(x₂ - x₁)。

**易错点**：斜率不存在时，直线垂直于 x 轴（倾斜角 90°）；斜率公式与两点顺序无关。

---

## 直线方程的五种形式

| 形式 | 方程 | 适用条件 |
|------|------|----------|
| 点斜式 | y - y₀ = k(x - x₀) | 斜率 k 存在 |
| 斜截式 | y = kx + b | 斜率 k 存在 |
| 两点式 | (y - y₁)/(y₂ - y₁) = (x - x₁)/(x₂ - x₁) | x₁≠x₂, y₁≠y₂ |
| 截距式 | x/a + y/b = 1 | a≠0, b≠0（不垂直于坐标轴且不过原点） |
| 一般式 | Ax + By + C = 0 (A²+B²≠0) | 表示所有直线 |

**易错点**：选用截距式时要注意截距为零的情况，需单独讨论。

---

## 两条直线的位置关系

设 l₁: y = k₁x + b₁ 或 A₁x + B₁y + C₁ = 0，l₂: y = k₂x + b₂ 或 A₂x + B₂y + C₂ = 0。

- **平行**：k₁ = k₂ 且 b₁ ≠ b₂（或 A₁B₂ = A₂B₁ 且 A₁C₂ ≠ A₂C₁）。  
- **重合**：k₁ = k₂ 且 b₁ = b₂（或 A₁B₂ = A₂B₁ 且 A₁C₂ = A₂C₁）。  
- **垂直**：k₁·k₂ = -1（或 A₁A₂ + B₁B₂ = 0）。  
- **相交**：斜率不相等。

**交点求解**：联立方程组。

---

## 距离公式

- **两点间距离**：|P₁P₂| = √[(x₂ - x₁)² + (y₂ - y₁)²]。  
- **点到直线距离**：点 P(x₀,y₀) 到直线 Ax+By+C=0 的距离 d = |Ax₀ + By₀ + C| / √(A² + B²)。  
- **平行线间距离**：l₁: Ax+By+C₁=0 与 l₂: Ax+By+C₂=0 的距离 d = |C₁ - C₂| / √(A² + B²)。

**易错点**：使用平行线距离公式前必须将 x、y 系数化为相同。

---

## 圆的方程

**标准方程**：(x - a)² + (y - b)² = r²，圆心 (a,b)，半径 r。  
**一般方程**：x² + y² + Dx + Ey + F = 0 (D²+E²-4F>0)，圆心 (-D/2, -E/2)，半径 ½√(D²+E²-4F)。

**点与圆的位置关系**：点到圆心距离与半径比较。  
**待定系数法**：通常设标准方程（已知圆心或半径）或一般方程（已知三点）。

**易错点**：二元二次方程表示圆需满足：x² 与 y² 系数相等且不为 0，无 xy 项，且 D²+E²-4F>0。

---

## 直线与圆的位置关系

设圆心到直线的距离为 d，圆半径为 r。

- 相交 ⇔ d < r（弦长 = 2√(r² - d²)）。  
- 相切 ⇔ d = r。  
- 相离 ⇔ d > r。

**切线问题**：  
- 过圆上一点求切线：利用圆心与切点连线垂直于切线（法向量）。  
- 过圆外一点求切线：设点斜式，利用圆心到直线距离等于半径求斜率，需讨论斜率不存在的情况。

**易错点**：过圆外一点引切线有两条，若只解出一个斜率，说明另一条斜率不存在。

## 椭圆及其标准方程

**定义**：平面内到两定点 F₁, F₂ 距离之和为常数 2a（大于 |F₁F₂| = 2c）的点的轨迹。

**标准方程**：  
- 焦点在 x 轴：x²/a² + y²/b² = 1 (a > b > 0, a² = b² + c²)。  
- 焦点在 y 轴：y²/a² + x²/b² = 1 (a > b > 0)。

**基本量**：长轴长 2a，短轴长 2b，焦距 2c，离心率 e = c/a (0 < e < 1)。

**易错点**：先判断焦点所在轴再写方程；椭圆上的点到焦点距离最大为 a+c，最小为 a-c。

---

## 双曲线及其标准方程

**定义**：平面内到两定点 F₁, F₂ 距离之差的绝对值为常数 2a（小于 |F₁F₂|）的点的轨迹。

**标准方程**：  
- 焦点在 x 轴：x²/a² - y²/b² = 1 (c² = a² + b²)。  
- 焦点在 y 轴：y²/a² - x²/b² = 1。

**基本量**：实轴长 2a，虚轴长 2b，焦距 2c，离心率 e = c/a (e > 1)。  
**渐近线**：焦点在 x 轴时 y = ±(b/a)x；焦点在 y 轴时 y = ±(a/b)x。

**易错点**：双曲线定义中绝对值不可丢；渐近线方程与焦点位置相关，勿混淆系数。

---

## 抛物线及其标准方程

**定义**：平面内到定点 F（焦点）与到定直线 l（准线）距离相等的点的轨迹。

**标准方程**（p > 0，p 为焦点到准线的距离）：

| 方程 | 焦点 | 准线 | 开口 |
|------|------|------|------|
| y² = 2px | (p/2, 0) | x = -p/2 | 向右 |
| y² = -2px | (-p/2, 0) | x = p/2 | 向左 |
| x² = 2py | (0, p/2) | y = -p/2 | 向上 |
| x² = -2py | (0, -p/2) | y = p/2 | 向下 |

**焦半径**：抛物线上一点到焦点的距离，等于该点到准线的距离。

**易错点**：一次项决定对称轴，系数正负决定开口方向。

## 直线与圆锥曲线的位置关系

**一般方法**：联立直线与曲线方程，消元得一元二次方程，用判别式 Δ 判断。

- Δ > 0：相交（两个交点）。  
- Δ = 0：相切（一个交点，抛物线/双曲线需注意平行于渐近线/对称轴的情况）。  
- Δ < 0：相离。

**弦长公式**：|AB| = √(1+k²) · |x₁ - x₂| = √(1+k²) · √[(x₁+x₂)² - 4x₁x₂]（k 为直线斜率）。  
亦可表示为 √(1+1/k²) · |y₁ - y₂|。

**易错点**：对双曲线，当直线与渐近线平行时，只有一个交点但非相切；对抛物线，平行于对称轴也只有一个交点。

---

## 圆锥曲线的统一定义与离心率

圆锥曲线可统一定义：平面内到一个定点 F（焦点）和一条定直线 l（准线）的距离之比为常数 e 的点的轨迹。  
- 0 < e < 1：椭圆。  
- e = 1：抛物线。  
- e > 1：双曲线。

**离心率的几何意义**：刻画曲线的“扁圆”或“开口”程度。椭圆 e 越接近 1 越扁，双曲线 e 越接近 1 开口越小。

**求离心率范围/值的方法**：  
- 直接求出 a、c 或 a、b 关系。  
- 利用几何约束转化为不等式。  
- 结合焦点三角形、渐近线等性质。

---

## 圆锥曲线的中点弦问题

**点差法**：已知弦中点坐标，求弦所在直线的斜率。

**步骤**：设弦两端点坐标代入曲线方程 → 两式相减 → 利用平方差公式 → 结合中点坐标求出斜率。

**适用范围**：椭圆、双曲线、抛物线内部中点弦。  
**易错点**：求出斜率后必须验证直线是否与曲线有两个交点（判别式 Δ > 0），否则不存在所求中点弦。

---

## 轨迹方程的五种求法

1. **直译法**：将几何条件直接转化为坐标等式。  
2. **定义法**：根据已知曲线的定义直接写出方程。  
3. **相关点法（代入法）**：被动点坐标用主动点坐标表示，代入主动点所在方程。  
4. **参数法**：引入参数，建立坐标间关系，消参得普通方程。  
5. **交轨法**：求两动曲线交点轨迹时，消去参数得到关于 x,y 的方程。

**易错点**：轨迹方程必须删除不合题意的点（即检验纯粹性与完备性）。
