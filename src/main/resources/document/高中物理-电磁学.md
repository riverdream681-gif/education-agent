# 高中物理 - 电磁学

---

## 电荷守恒定律与库仑定律

**电荷守恒定律**：电荷既不能被创造，也不能被消灭，只能从一个物体转移到另一个物体，或者从物体的一部分转移到另一部分；在转移过程中，电荷的总量保持不变。

**元电荷**：e = 1.60×10⁻¹⁹ C，所有带电体的电荷量都是 e 的整数倍。

**库仑定律**：真空中两个静止点电荷之间的相互作用力，与它们的电荷量的乘积成正比，与它们的距离的二次方成反比，作用力的方向在它们的连线上。  
公式：F = k·Q₁Q₂/r²，k = 9.0×10⁹ N·m²/C²。

**适用条件**：真空、静止、点电荷。

**易错点**：点电荷是理想模型，带电体能否视为点电荷由问题情景决定；计算时电荷量不带正负号，方向根据“同斥异吸”判断。

## 电场强度与电场线

**电场强度**：试探电荷在电场中某点所受的电场力 F 与它的电荷量 q 的比值，E = F/q（定义式，矢量，方向与正电荷受力方向相同）。  
**决定式**：点电荷电场 E = kQ/r²；匀强电场 E = U/d。

**电场线**：用来形象描述电场分布的一簇曲线。特点：始于正电荷（或无穷远），终于负电荷（或无穷远）；不闭合、不相交；疏密表示场强大小；切线方向表示该点场强方向。

**电场的叠加**：多个点电荷在某点的合场强为各场强的矢量和，遵循平行四边形定则。

**易错点**：电场强度由场源电荷和该点位置决定，与试探电荷无关；电场线不是电荷的运动轨迹。

## 电势能与电势

**电势能**：电荷在电场中具有的势能。电场力做正功，电势能减少；电场力做负功，电势能增加。W_电 = -ΔE_p。

**电势**：电荷在电场中某点的电势能与其电荷量的比值，φ = E_p/q（标量）。沿电场线方向电势逐渐降低。电势具有相对性，通常取无穷远处或大地为零电势。

**常见场电势分布**：正点电荷周围 φ > 0，负点电荷周围 φ < 0；匀强电场中电势均匀变化。

**易错点**：电势能大小与电荷正负有关，正电荷在电势高处电势能大，负电荷反之；电势高低与试探电荷无关。

---

## 电势差与等势面

**电势差（电压）**：电场中两点间电势的差值，U_AB = φ_A - φ_B，与零电势点选择无关。W_AB = qU_AB。

**等势面**：电场中电势相等的点构成的曲面。特点：同一等势面上移动电荷电场力不做功；等势面与电场线垂直；电场线由高电势等势面指向低电势等势面；等势面越密处场强越大。

**易错点**：电势差是标量，其正负只表示两点电势高低关系；U_AB = -U_BA；W=qU 中 q、U、W 均带正负号运算。

## 电容器的电容

**电容器**：两个彼此绝缘又相距很近的导体组成的储存电荷的元件。  
**电容**：电容器所带电荷量 Q 与两极板间电势差 U 的比值，C = Q/U（定义式）。单位：法拉（F），常用 μF、pF。

**平行板电容器的决定式**：C = εᵣS/(4πkd)（真空中 C = S/(4πkd)），电容由极板正对面积 S、板间距 d、电介质的相对介电常数 εᵣ 决定。

**动态分析**：  
- 充电后与电源断开：Q 不变，由 U = Q/C，E = U/d 等分析。  
- 始终与电源连接：U 不变，由 Q = CU，E = U/d 等分析。

**易错点**：定义式 C=Q/U 中 C 与 Q、U 无关，由电容器本身结构决定；注意区分“电容变化”和“电荷量变化”两类问题。

## 欧姆定律与串并联电路

**欧姆定律**：导体中的电流 I 跟导体两端的电压 U 成正比，跟导体的电阻 R 成反比，I = U/R（适用于纯电阻电路、线性元件）。

**串并联电路特点**：  
| 连接方式 | 电流 | 电压 | 总电阻 |
|----------|------|------|--------|
| 串联 | I = I₁ = I₂ | U = U₁ + U₂ | R = R₁ + R₂ |
| 并联 | I = I₁ + I₂ | U = U₁ = U₂ | 1/R = 1/R₁ + 1/R₂ |

**电压表、电流表的改装**：电流表串联大电阻改电压表；并联小电阻改电流表。

**易错点**：电阻的阻值由本身性质决定，与 U、I 无关；I = U/R 中 U、I、R 必须是同一段电路的对应值。

## 电功、电功率与焦耳定律

**电功**：电场力移动电荷做功，W = UIt（适用于任何电路）。  
**电功率**：P = W/t = UI（任何电路）。

**焦耳定律**：电流通过导体产生的热量 Q = I²Rt（适用于任何电路）。  
**热功率**：P_热 = I²R。

**纯电阻电路**（电能全部转化为内能）：W = Q = UIt = I²Rt = U²t/R，P = P_热 = UI = I²R = U²/R。  
**非纯电阻电路**（含电动机、电解槽等）：W = UIt > Q = I²Rt，UIt = I²Rt + E_其他。

**易错点**：电动机的额定电压、额定电流不等于线圈电阻直接决定；非纯电阻电路不能用 U/R 计算电流，也不可用 U²/R 算总功率。

## 闭合电路欧姆定律

**电动势 E**：电源把其他形式的能转化为电能的本领，数值上等于非静电力把 1C 正电荷从负极移送到正极所做的功。

**闭合电路欧姆定律**：闭合电路中的电流与电源电动势成正比，与内外电阻之和成反比，I = E/(R+r)。  
**路端电压**：U = E - Ir（随外电阻增大而增大；外电路断开时 U = E；短路时 U = 0, I = E/r）。

**功率关系**：电源总功率 P_总 = EI；内耗功率 P_内 = I²r；输出功率 P_出 = UI = I²R = E²R/(R+r)²（当 R = r 时输出功率最大 P_m = E²/(4r)）。  
**电源效率**：η = P_出/P_总 ×100% = U/E ×100% = R/(R+r) ×100%。

**易错点**：路端电压 U = E - Ir 中的 U 是外电路总电压，不是某一元件电压；纯电阻电路可以用 U = IR。

## 磁场的描述与安培力

**磁场**：磁体或电流周围存在的特殊物质。  
**磁感应强度 B**：描述磁场强弱和方向的物理量，B = F/(IL)（I⊥B 时），矢量，方向即小磁针 N 极受力方向。

**磁感线**：闭合曲线，外部从 N 到 S，内部从 S 到 N；疏密表示 B 大小；切线方向表示 B 方向。  
**常见磁场**：条形磁铁、蹄形磁铁、直线电流（安培定则）、环形电流、通电螺线管。

**安培力**：通电导线在磁场中受到的力。大小 F = BIL sinθ（θ 为 B 与 I 夹角），方向用左手定则判断（F⊥B，F⊥I，即 F⊥B 与 I 决定的平面）。

**易错点**：B 与试探电流无关；安培力公式中 L 是有效长度；左手定则中四指指向电流方向，掌心让磁感线穿入。

## 洛伦兹力与带电粒子在磁场中的运动

**洛伦兹力**：运动电荷在磁场中受到的力。大小 f = qvB sinθ（θ 为 v 与 B 夹角），方向由左手定则判断（正电荷四指指向 v 方向，负电荷相反）。  
**特点**：洛伦兹力始终与速度方向垂直，永不做功，只改变速度方向不改变大小。

**带电粒子在匀强磁场中的匀速圆周运动**（v⊥B）：  
由 qvB = mv²/r，得轨道半径 r = mv/(qB)；周期 T = 2πm/(qB)（与速度大小无关）。

**解题要点**：画轨迹、定圆心、求半径、找圆心角（θ=α，时间 t = (θ/2π)T 或 t = θm/(qB)）。  
**圆心确定法**：速度垂线交点；或弦的中垂线与速度垂线交点。

**易错点**：洛伦兹力只改变方向不做功；周期与速度无关，但前提是垂直于 B 分量不变；回旋加速器中交变电压周期等于粒子在磁场中的周期。

## 电磁感应现象与楞次定律

**电磁感应现象**：穿过闭合电路的磁通量发生变化时，电路中产生感应电动势和感应电流的现象。

**产生感应电流的条件**：① 电路闭合；② 穿过电路的磁通量 Φ 发生变化（Φ = BS sinθ）。

**楞次定律**：感应电流的磁场总要阻碍引起感应电流的磁通量的变化。  
**应用步骤**：明确定原磁场方向及 Φ 变化 → 判“增反减同”确定感应电流磁场方向 → 安培定则定感应电流方向。

**右手定则**：用于判定导体切割磁感线产生感应电流的方向（伸开右手，拇指与四指垂直，磁感线穿入手心，拇指指向导体运动方向，四指指向感应电流方向）。楞次定律的推论：“来拒去留”“增缩减扩”。

**易错点**：阻碍的是“磁通量的变化”而不是原磁场本身；阻碍不是阻止，最终变化仍在发生。

## 法拉第电磁感应定律

**内容**：闭合电路中感应电动势的大小，跟穿过这一电路的磁通量的变化率成正比。E = n·ΔΦ/Δt（n 为线圈匝数）。

**导线切割磁感线产生的感应电动势**：E = BLv sinθ（θ 为 B 与 v 夹角；L 为有效切割长度，B、L、v 两两垂直时 E = BLv）。  
**转动切割**：E = ½BωL²（导体棒绕一端垂直于 B 的匀强磁场中转动）。

**感应电动势与感应电流**：只有电路闭合时才有感应电流，但不管电路是否闭合，只要磁通量变化就有感应电动势。可以将产生感应电动势的部分看作电源，电动势方向用楞次定律或右手定则判定。

**易错点**：ΔΦ/Δt 是变化率，与 Φ、ΔΦ 无直接大小关系；在 Φ-t 图像中，E 与斜率成正比；E=BLv 中 v 为瞬时速度时 E 为瞬时值。

## 自感与涡流

**自感现象**：由于线圈自身的电流发生变化而产生电磁感应的现象。自感电动势 E = L·ΔI/Δt，L 为自感系数，与线圈匝数、形状、有无铁芯有关。

**通电自感与断电自感**：通电时，自感线圈阻碍电流增大，灯泡延迟变亮（串联）；断电时，自感线圈阻碍电流减小，灯泡可能闪亮后熄灭（条件：原线圈电流大于灯泡原电流）。

**涡流**：块状金属处在变化的磁场中或相对于磁场运动时，金属内部产生的旋涡状感应电流。热效应应用（高频感应炉）、危害（变压器铁芯用硅钢片叠成以减少涡流损耗）。

**易错点**：断电自感中灯泡“闪亮”需满足线圈原来电流大于灯泡原来电流；电感线圈电流不能突变。

## 交变电流

**交变电流（AC）**：大小和方向都随时间做周期性变化的电流。  
**正弦式交变电流**：矩形线圈在匀强磁场中绕垂直于磁场的轴匀速转动时产生。  
瞬时值：e = E_m sinωt，u = U_m sinωt，i = I_m sinωt（从中性面开始计时）。  
峰值：E_m = nBSω。  
有效值：根据电流热效应定义，正弦交流电有效值 E = E_m/√2，U = U_m/√2，I = I_m/√2（仅对正弦式成立）。  
周期、频率：T = 1/f，ω = 2π/T = 2πf。

**“四值”用途**：峰值（电容耐压）、有效值（电表示数、功率、熔断电流）、平均值（计算电荷量 q = nΔΦ/R）、瞬时值（分析某一时刻）。

**易错点**：只有正弦式交变电流有效值才等于最大值的 1/√2；电表示数、用电器铭牌均为有效值。

## 理想变压器与远距离输电

**理想变压器**：无能量损耗，P_入 = P_出。  
电压关系：U₁/U₂ = n₁/n₂（与匝数成正比）；  
电流关系：I₁/I₂ = n₂/n₁（一原一副时，与匝数成反比）；  
频率关系：f₁ = f₂（变压器不改变频率）。  
**多个副线圈**：P₁ = P₂ + P₃ + …；U₁/n₁ = U₂/n₂ = U₃/n₃；n₁I₁ = n₂I₂ + n₃I₃ + …。

**远距离输电**：减小输电线上功率损耗（P_损 = I²R 线）的关键是提高输电电压 U 输。  
升压变压器 → 高压输电 → 降压变压器。  
功率关系：P_发 = P_损 + P_用；电压关系：U_升/U_输 = n₁/n₂，U_输 = U_损 + U_降。

**易错点**：变压器只能改变交流电压，不能改变恒定直流；原线圈输入电压决定副线圈输出电压；副线圈输出功率决定原线圈输入功率。

## 电磁振荡与电磁波

**电磁振荡**：在 LC 振荡电路中，电容器极板上的电荷 q、电路中的电流 i、电场能、磁场能均随时间做周期性变化。  
周期公式（固有周期）：T = 2π√(LC)，频率 f = 1/(2π√(LC))。  
振荡过程：电容放电时电场能→磁场能，充电时磁场能→电场能，无阻尼振荡总能量守恒。

**电磁波**：变化的电场和磁场由近及远地向周围传播形成电磁波。特点：横波，在真空中传播速度 c = 3.0×10⁸ m/s；E 与 B 相互垂直且均与传播方向垂直；v = λf，c = λf（真空中）；不需要介质；具有反射、折射、干涉、衍射等现象。

**麦克斯韦电磁场理论**：变化的磁场产生电场，变化的电场产生磁场。均匀变化产生稳定场，非均匀变化产生变化的场。

**易错点**：LC 回路中振荡电流最大时电容器电压为零，放电完毕；恒定磁场不能产生电场。
