# 高中物理 - 力学

---

## 质点、参考系和坐标系

**质点**：用来代替物体的有质量的点，是一种理想化模型。物体可视为质点的条件：物体的大小和形状对所研究问题的影响可以忽略。

**参考系**：描述物体运动时选作参考的物体。同一运动，选取不同参考系，运动状态可能不同，一般默认以地面为参考系。

**坐标系**：定量描述物体位置及位置变化。常见有一维直线坐标系、二维平面直角坐标系。

**易错点**：质点不一定是很小的物体，很大物体也可视为质点；参考系的选择是任意的，但应以描述方便为原则。

---

## 位移与路程

**位移**：从初位置指向末位置的有向线段，是矢量，大小等于初末位置间的直线距离，方向由初位置指向末位置。

**路程**：物体运动轨迹的长度，是标量。

**关系**：位移大小 ≤ 路程。单向直线运动中位移大小等于路程，其他情况下路程大于位移大小。

**易错点**：位移只与初末位置有关，与路径无关；描述位移时必须同时指明大小和方向。

---

## 速度与加速度

**速度**：描述运动快慢和方向的物理量。平均速度 v = Δx/Δt（矢量，方向与位移方向一致）；瞬时速度即某时刻或某位置的速度，方向即运动方向。

**速率**：瞬时速度的大小，是标量；平均速率 = 路程/时间。

**加速度**：描述速度变化快慢的物理量，a = Δv/Δt，矢量，方向与速度变化量 Δv 方向相同。当 a 与 v 同向时加速，反向时减速。

**易错点**：速度大不等于加速度大；加速度减小不表示速度减小；加速度方向不一定与速度方向共线（曲线运动）。

---

## 匀变速直线运动的基本公式

**适用条件**：加速度 a 恒定不变的直线运动。

**基本公式**（初速度 v₀，末速度 v，加速度 a，时间 t，位移 x）：
1. v = v₀ + at  
2. x = v₀t + ½at²  
3. v² - v₀² = 2ax  
4. x = ½(v₀ + v)t

**推论**：  
- 中间时刻速度 v_{t/2} = ½(v₀ + v) = 平均速度。  
- 相邻相等时间间隔位移差 Δx = aT²。

**易错点**：先规定正方向，与正方向相反的物理量取负值；刹车问题要先判断停止时间。

---

## 匀变速直线运动的重要推论与比例式

**初速度为零的匀加速直线运动（v₀=0）**：

- **按时间等分**：  
  1T 末、2T 末、3T 末……的速度比 v₁:v₂:v₃:… = 1:2:3:…  
  1T 内、2T 内、3T 内……的位移比 x₁:x₂:x₃:… = 1:4:9:…  
  第 1T 内、第 2T 内、第 3T 内……的位移比 xⅠ:xⅡ:xⅢ:… = 1:3:5:…

- **按位移等分**：  
  通过连续相等的位移所用的时间比 t₁:t₂:t₃:… = 1:(√2-1):(√3-√2):…

**易错点**：必须确认初速度为零；匀减速至零可视为反向的初速度为零的匀加速，适用比例式。

## 自由落体运动与竖直上抛运动

**自由落体运动**：v₀=0，a=g，只受重力。公式：v = gt，h = ½gt²，v² = 2gh。

**竖直上抛运动**：初速度竖直向上，a = -g。  
- 分段法：上升过程匀减速，下降过程自由落体。  
- 整体法：取竖直向上为正，v = v₀ - gt，x = v₀t - ½gt²，v² - v₀² = -2gx。  
- 对称性：时间对称（上升与下降同一高度时间相等），速度对称（同一高度速度大小相等、方向相反）。

**易错点**：自由落体是从静止开始下落，不是单纯的匀加速下落；竖直上抛最高点 v=0 但 a=g 不为零。

---

## 重力、弹力、摩擦力

**重力**：G = mg，方向竖直向下，作用点在重心。重心不一定在物体上。

**弹力**：产生条件——接触、发生弹性形变。方向与施力物体形变方向相反。常见弹力：支持力（垂直接触面）、绳拉力（沿绳收缩方向）、弹簧弹力（胡克定律 F = kx）。

**摩擦力**：  
- 静摩擦力：与相对运动趋势方向相反，大小 0 < F ≤ F_max，需由平衡条件或牛顿第二定律求解。  
- 滑动摩擦力：F_f = μF_N，方向与相对运动方向相反。  
- 最大静摩擦力略大于滑动摩擦力，一般可认为 F_max ≈ μF_N。

**易错点**：有弹力不一定有摩擦力，有摩擦力必有弹力；摩擦力方向与物体运动方向可能相同（如人走路）。

## 力的合成与分解

**合力与分力**：等效替代关系。力的合成与分解遵循**平行四边形定则**（或三角形定则）。

**合力范围**：|F₁ - F₂| ≤ F_合 ≤ F₁ + F₂。

**分解方法**：通常按力的实际作用效果分解，或采用正交分解法（建立直角坐标系，将各力沿坐标轴分解）。

**正交分解步骤**：建系 → 分解 → 列平衡方程（或牛二方程）ΣF_x = ma_x，ΣF_y = ma_y。

**易错点**：合力可能比分力小；分解时平行四边形中实线表示力，虚线表示辅助线。

---

## 共点力作用下物体的平衡

**平衡条件**：物体所受合力为零，即 ΣF = 0。在正交分解下：ΣF_x = 0，ΣF_y = 0。

**常见解法**：
1. 合成法：将多力转化为二力平衡。  
2. 分解法：正交分解列方程。  
3. 矢量三角形法：三力平衡时，三力平移构成封闭三角形；若动态变化，常用图解法。  
4. 相似三角形法：力的矢量三角形与实际几何三角形相似。

**易错点**：明确研究对象，正确受力分析（一重二弹三摩擦四其他）；多物体问题优先整体法，再隔离法。

---

## 牛顿第一定律与惯性

**牛顿第一定律**：一切物体总保持匀速直线运动状态或静止状态，直到有外力迫使它改变这种状态为止。

**惯性**：物体保持原来运动状态不变的性质。质量是惯性大小的唯一量度。一切物体在任何状态下都有惯性。

**易错点**：力是改变运动状态的原因，不是维持运动的原因；惯性不是力，不能说“受到惯性”或“惯性力”。

---

## 牛顿第二定律

**内容**：物体加速度的大小跟它受到的作用力成正比，跟它的质量成反比，加速度的方向跟作用力的方向相同。  
**表达式**：F_合 = ma。

**特性**：矢量性、瞬时性（a 与 F 合同时变化）、独立性（各力独立产生加速度）、同体性（F_合、m、a 对应同一物体）。

**解题步骤**：
1. 明确研究对象。  
2. 受力分析与运动状态分析。  
3. 建立坐标系（常以加速度方向为正）。  
4. 列方程 ΣF_x = ma_x，ΣF_y = ma_y。  
5. 求解并讨论。

**易错点**：必须求合力；不能将某个分力直接代入 F_合 位置；国际单位制中 F 取 N，m 取 kg，a 取 m/s²。

---

## 牛顿第三定律

**内容**：两物体间的作用力和反作用力总是大小相等、方向相反、作用在同一条直线上。

**特点**：同值、同线、同性（同种性质的力）、同时产生同时消失、分别作用在两个物体上。

**一对作用力与反作用力 vs 一对平衡力**：  
- 作用力与反作用力：作用在不同物体上，不可抵消，性质相同。  
- 平衡力：作用在同一物体上，合力为零，性质不一定相同。

**易错点**：鸡蛋碰石头，两者受力大小相等，只是承受力能力不同。

---

## 超重与失重

**超重**：物体对支持物的压力（或对悬挂物的拉力）大于重力的现象。加速度方向向上，可能加速上升或减速下降。

**失重**：物体对支持物的压力（或拉力）小于重力的现象。加速度方向向下，可能加速下降或减速上升。

**完全失重**：a = g 且方向竖直向下，物体对支持物、悬挂物完全没有作用力（如自由落体、卫星绕行）。

**易错点**：超重、失重不是重力变化，而是“视重”变化，实际重力 mg 不变。

---

## 运动的合成与分解（小船渡河、绳杆末端速度）

**合运动与分运动**：具有等时性、独立性、等效性。遵循平行四边形定则。

**小船渡河**：  
- 最短时间：船头始终垂直河岸，t_min = d / v_船。  
- 最短位移：若 v_船 > v_水，可垂直渡河，位移为河宽 d；若 v_船 < v_水，位移大于河宽，需利用矢量圆求解。

**绳（杆）末端速度分解**：沿绳（杆）方向的速度分量相等；常将末端速度分解为沿绳方向和垂直于绳方向。

**易错点**：合速度是实际运动速度；船头方向与船速方向一致，合速度为船速与水速矢量和。

---

## 平抛运动

**特点**：初速度水平，仅受重力作用，加速度恒为 g 的匀变速曲线运动。

**分解**：  
- 水平方向：匀速直线运动，v_x = v₀，x = v₀t。  
- 竖直方向：自由落体运动，v_y = gt，y = ½gt²。  
- 合速度 v = √(v_x² + v_y²)，速度偏转角 θ 满足 tanθ = v_y/v_x。  
- 合位移 s = √(x² + y²)，位移偏转角 α 满足 tanα = y/x，且 tanθ = 2tanα。

**飞行时间** t = √(2h/g)，只由高度决定。

**易错点**：平抛是匀变速曲线运动，相等时间内速度变化量 Δv = gΔt，方向竖直向下。

---

## 圆周运动

**描述物理量**：线速度 v，角速度 ω，周期 T，频率 f，向心加速度 a_n。关系：v = ωr = 2πr/T，a_n = v²/r = ω²r = 4π²r/T²。

**匀速圆周运动**：v 大小不变，方向时刻变化；向心加速度指向圆心，由合外力提供向心力 F_合 = ma_n。

**非匀速圆周运动**：合外力不指向圆心，一般分解为切向力和法向力。

**常见模型**：水平圆盘、圆锥摆、竖直面内的绳模型与杆模型（最高点临界速度）。  
- 绳模型：最高点最小速度 v_min = √(gr)。  
- 杆模型：最高点最小速度 v_min = 0。

**易错点**：向心力是按效果命名的力，不是额外受到的力，受力分析时不能说“受到向心力”。
