# 高中数学 - 函数与导数

---

## 导数的几何意义

函数 f(x) 在点 x₀ 处的导数 f'(x₀) 表示曲线 y=f(x) 在点 (x₀, f(x₀)) 处切线的斜率。  
切线方程公式：y - f(x₀) = f'(x₀)(x - x₀)。

**常见题型**：
1. 已知函数表达式，求某点处的切线方程
2. 已知切线斜率或切线过某点，反求参数

**易错点**：求切线方程时需先确认切点是否在曲线上。

---

## 利用导数研究函数的单调性

若在区间 (a, b) 内 f'(x) > 0，则 f(x) 在该区间单调递增；  
若 f'(x) < 0，则 f(x) 在该区间单调递减。

**求解步骤**：
1. 求定义域
2. 求 f'(x)
3. 解不等式 f'(x) > 0 和 f'(x) < 0
4. 写出单调区间

---

## 利用导数求函数的极值与最值

极值点判定：f'(x₀) = 0 且 f'(x) 在 x₀ 两侧变号。

**求解步骤**：
1. 求 f'(x)
2. 令 f'(x) = 0，解出驻点
3. 用符号表判断各驻点是否为极值点
4. 结合端点值求最值

**易错点**：f'(x₀) = 0 是极值点的必要非充分条件，如 y = x³ 在 x=0 处导数为 0 但无极值。

## 函数的概念与表示

设 A, B 是非空数集，若按照某种确定的对应关系 f，使对于集合 A 中的任意一个数 x，在集合 B 中都有唯一确定的数 f(x) 与之对应，则称 f: A→B 为从集合 A 到集合 B 的一个函数。

**三要素**：定义域、值域、对应法则。  
两个函数相等当且仅当定义域和对应法则都相同。

**常见题型**：判断同一函数；求抽象函数定义域；求函数解析式。  
**易错点**：函数定义域必须写成集合或区间的形式，且要写成最简结果。

---

## 函数的定义域与值域

**定义域**：自变量 x 的取值范围。常见限制：  
- 分母不为 0  
- 偶次根式被开方数 ≥ 0  
- 对数真数 > 0  
- 零指数幂的底数 ≠ 0  
- 实际问题中的实际意义

**值域**：因变量 y 的取值范围。常见求法：观察法、配方法、换元法、判别式法、单调性法、导数法等。

**易错点**：变形解析式时定义域可能改变（如约分消去公因式），求解时需回归原函数定义域。

---

## 函数的单调性（定义法）

设函数 f(x) 定义域为 I，区间 D ⊆ I：  
若 ∀x₁,x₂ ∈ D，当 x₁ < x₂ 时，总有 f(x₁) < f(x₂)，则 f(x) 在 D 上单调递增；  
若 ∀x₁,x₂ ∈ D，当 x₁ < x₂ 时，总有 f(x₁) > f(x₂)，则 f(x) 在 D 上单调递减。

**判断方法**：定义法（取值、作差、变形、定号、结论）；图像法；复合函数“同增异减”。  
**常见题型**：证明单调性；求单调区间；利用单调性解不等式、比较大小。  
**易错点**：单调区间不能并在一起，要用“和”或逗号分隔。

---

## 函数的奇偶性

**奇函数**：定义域关于原点对称，且 ∀x，f(-x) = -f(x)，图像关于原点对称。  
**偶函数**：定义域关于原点对称，且 ∀x，f(-x) = f(x)，图像关于 y 轴对称。

**常用性质**：  
- 奇函数在对称区间上单调性相同，偶函数相反。  
- 若奇函数在 x = 0 有定义，则 f(0) = 0。  
- 既奇又偶的函数只能是 f(x) = 0（定义域对称）。

**易错点**：必须先验证定义域是否关于原点对称，再验证等式。

---

## 函数的周期性

对于函数 y = f(x)，若存在非零常数 T，使得定义域内的任意 x 都有 f(x+T) = f(x)，则称 f(x) 为周期函数，T 为其周期。最小正周期称为函数的基本周期。

**常见周期函数**：  
- 三角函数：y = sin x, y = cos x 的周期为 2π；y = tan x 的周期为 π。  
- 一般抽象函数：f(x+a) = f(x) 表示周期为 |a|；f(x+a) = -f(x) 表示周期为 2|a|。

**常见题型**：求周期；利用周期性求值；判断周期函数图像特征。

## 基本初等函数（一）

**指数函数 y = aˣ (a>0 且 a≠1)**  
- 定义域 R，值域 (0, +∞)，图像恒过 (0,1)。  
- 当 a>1 时单调递增；当 0<a<1 时单调递减。

**对数函数 y = logₐx (a>0 且 a≠1)**  
- 定义域 (0, +∞)，值域 R，图像恒过 (1,0)。  
- 当 a>1 时单调递增；当 0<a<1 时单调递减。  
- 与指数函数互为反函数，图像关于直线 y=x 对称。

**幂函数 y = xᵅ (α∈R)**  
- 图像恒过 (1,1)。  
- α>0 时在 (0,+∞) 上递增；α<0 时递减。  
- 常见图像：α=1,2,3,1/2,-1 等需熟记。

**常见题型**：比较大小；解指对不等式；图像识别与定点问题。

## 导数的运算

**基本初等函数的导数公式**  
C' = 0；(xⁿ)' = n xⁿ⁻¹；(sinx)' = cosx；(cosx)' = -sinx；  
(eˣ)' = eˣ；(aˣ)' = aˣ ln a；(lnx)' = 1/x；(logₐx)' = 1/(x ln a)。

**导数的四则运算法则**  
(u ± v)' = u' ± v'；(uv)' = u'v + uv'；(u/v)' = (u'v - uv') / v² (v≠0)。

**复合函数求导法则**（链式法则）  
设 y = f(g(x))，则 y' = f'(g(x))·g'(x)。

**易错点**：区分复合与乘积运算；能化简的先化简再求导；注意定义域对导数的影响。

## 利用导数研究函数的零点与方程根

函数零点 ⇔ 方程 f(x)=0 的实数根 ⇔ 函数图像与 x 轴交点的横坐标。

**常用方法**：  
1. 直接解方程（可解时）。  
2. 零点存在定理：若 f(x) 在 [a,b] 连续且 f(a)f(b)<0，则在 (a,b) 内至少有一个零点。  
3. 结合导数分析单调性与极值，绘制草图，讨论零点个数。

**易错点**：  
- 零点存在定理只能判定存在性，不能确定唯一性，需结合单调性。  
- 极值点处函数值为零时，该零点是切点零点（重根），不计入变号零点个数。
